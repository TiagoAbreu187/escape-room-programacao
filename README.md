# Jogo de Aposta

História do jogo:  

Você ganhou 10 reais de presente no seu aniversário e precisa pagar as contas.  
Como 10 reais não são suficientes, você decide apostar!  
Sua missão é transformar 10 reais em 2 mil.  
Boa Sorte!

# Como executar?

Copie o código completo do arquivo Jogo.py e execute dentro do Visual Studio Code com Python instalado.
